Persyaratan

  • Persyaratan Data Diri Pemilik Bangunan Gedung

    Memilih jenis Status Kepemilikan Bangunan Gedung, yang dapat berupa Pemerintah, Badan Usaha atau Perseorangan.

    Melakukan pengisian Nomor Telepon/HP dan Alamat Email.

  • Persyaratan Data Alamat Bangunan Gedung

    Melakukan pengisian Alamat Bangunan Gedung termasuk di dalamnya informasi terkait Provinsi, Kab/Kota, Kecamatan serta Kelurahan Bangunan Gedung.

  • Persyaratan Data Bangunan Gedung

    Mendefinisikan Nama Bangunan.

    Mencantumkan informasi terkait Luas Bangunan.

    Mencantumkan informasi terkait Jumlah Lantai Bangunan.

    Mencantumkan informasi terkait Tinggi Bangunan.

    Mencantumkan informasi terkait Luas Basement Bangunan.

    Mencantumkan informasi terkait Jumlah Lantai Basement Bangunan.

    Mencantumkan informasi terkait Perancang Dokumen Teknis.

  • Persyaratan Data Tanah

    Mendefinisikan Jenis Dokumen Kepemilikan Data Tanah.

    Mencantumkan informasi terkait Tanggal Terbit Dokumen.

    Mencantumkan informasi terkait Hak Kepemilian Atas Tanah.

    Mencantumkan informasi terkait Alamat Lokasi Tanah.

    Mencantumkan informasi terkait File Data Kepemilikan Tanah.

    Mencantumkan informasi terkait Nomor Dokumen Data Tanah.

    Mencantumkan informasi terkait Luas Tanah (m2).

    Mencantumkan informasi terkait Nama Pemlik Hak Atas Tanah.

    Mencantumkan informasi terkait Perjanjian Tertulis Pemanfaatan Tanah.

    Mengunggah dokumen pendukung seperti Gambar dan Informasi tentang hasil penyelidikan tanah untuk bangunan sederhana serta Gambar dan Informasi tentang hasil penyelidikan tanah dengan format (*.pdf).

  • Persyaratan Data Teknis Struktur

    Mengunggah dokumen Gambar Situasi.

    Mengunggah dokumen Gambar Rencana Tapak.

    Mengunggah dokumen Gambar Denah.

    Mengunggah dokumen Gambar Potongan.

    Mengunggah dokumen Gambar Tampak.

    Mengunggah dokumen Gambar Detail Arsitektur dan Spesifikasi Teknis.

  • Persyaratan Data Teknis Struktur

    Mengunggah dokumen Gambar Rencana Fondasi.

    Mengunggah dokumen Gambar Rencana Kolom.

    Mengunggah dokumen Gambar Rencana Balok.

    Mengunggah dokumen Gambar Rencana Rangka Atap.

    Mengunggah dokumen Gambar Detail Struktur.

    Mengunggah dokumen Perhitungan Teknis Sederhana dan Gambar Rencana Plat Lantai.

  • Persyaratan Data Kelengkapan

    Mengunggah dokumen Gambar Rencana dan Perhitungan Teknis Jaringan Listrik.

    Mengunggah dokumen Gambar Rencana dan Perhitungan Sistem Sanitasi.
